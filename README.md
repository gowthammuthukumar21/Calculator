# 🧮 Simple Calculator

A basic and responsive **Calculator Web Application** built using **HTML, CSS, and JavaScript**.  
This project performs basic arithmetic operations with a clean user interface and interactive buttons.

---

## 🚀 Features

✨ **Basic Calculations**
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Percentage (%)

⌫ **Input Controls**
- Clear button to reset calculator
- Delete button to remove the last entered value

📜 **Calculation History**
- Stores previous calculations
- Displays calculation records
- Clear history option

⌨️ **Keyboard Support**
- Numbers and operators can be entered using keyboard
- Enter key to calculate
- Backspace to delete
- Escape key to clear

🎨 **Modern UI Design**
- Attractive gradient background
- Glassmorphism calculator layout
- Smooth hover animations
- Responsive design
  
---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage (for saving history)
